CLASSES OF ELEMENTS

    There are two classes of elements in HTML which include;

  1. Block Element
  2. Inline Element

DIFFRENCE BETWEEN THE TWO TYPES OF ELEMENTS

  • BLOCK ELEMENT

    • The Block ELement is an element that always start in a new line. Eg

    • p element
    • h1-h6 element
    • jjjjjjjj
  • INLINE ELEMENT

    • An Inline Element is an Element that will always merge with other element in the same line. It does not stand on its own
